## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Verified data structure, column types, and overall quality.

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions and trends.
* Identified missing values and outliers.
* Generated summary statistics.
* Created visualizations to understand key patterns.

### 3. Data Cleaning

* Removed duplicate records.

## How to Run

### Prerequisites

* Python 3.x
* PostgreSQL / MySQL / SQL Server
* Power BI Desktop

### Steps

1. Clone the repository.
2. Install required Python libraries:

   ```
   pip install pandas numpy matplotlib seaborn
   ```
3. Load the dataset into Python.
4. Run EDA and data cleaning scripts.
5. Execute SQL queries in your preferred database system.
6. Open the Power BI file to explore the dashboard.
7. Review the report and presentation for detailed insights.
